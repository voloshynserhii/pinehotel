# Room Images

Place room images here with the following naming convention:
- `double-1.jpg`, `double-2.jpg`, `double-3.jpg` - Double room images
- `suite-1.jpg`, `suite-2.jpg`, `suite-3.jpg` - Suite images  
- `family-1.jpg`, `family-2.jpg`, `family-3.jpg` - Family room images

Recommended:
- Format: JPEG or WebP
- Dimensions: min 1200x800px (aspect ratio 3:2)
- Optimized for web (quality 80-85%)
